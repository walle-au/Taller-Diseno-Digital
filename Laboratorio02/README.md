# Taller de Diseño Digital

## Grupo 3

Este repositorio corresponde al curso **EL3313 - Taller de Diseño Digital**.  
Está enfocado en el desarrollo y documentación de los laboratorios del curso, incluyendo diseño, simulación, síntesis y validación de circuitos digitales utilizando HDL (SystemVerilog/Verilog) y herramientas EDA.

En este segundo laboratorio se trabaja el diseño e implementación de un sistema empotrado basado en un microcontrolador RISC-V (RV32I), incluyendo memorias ROM y RAM, periféricos de entrada/salida (LEDs, switches, botones) y comunicación serie UART, todo implementado y validado en una FPGA Nexys4.
