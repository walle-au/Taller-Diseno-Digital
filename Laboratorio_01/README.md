# Laboratorio 1

En este laboratorio se introduce el diseño de circuitos digitales utilizando **SystemVerilog/Verilog**, así como el uso de herramientas de simulación y síntesis.

Se desarrollan ejercicios básicos como:

- Manejo de switches y LEDs.
- Diseño de un multiplexor 4:1 parametrizable.
- Generación de señal PWM.
- Implementación básica de una interfaz UART.

Incluye diseño en HDL, testbenches para validación y pruebas en FPGA.
